BASYS Board Calculator

Created a multi-mode switch controlled calculator capable of four bit adding, subtracting (with overflow indicator LED), comparator, and NAND operations. Programmed using VHDL in Vivado on the BASYS 3 board.

How It Works

Four different operations were able to be selected using a 2 bit input (2 switches on the BASYS board). All four operations were controlled by two four bit inputs (8 switches total).

The adder and subtractor works by cascading four one bit adders/subtractors, while displaying the result on a 7 segment display in Hex.

The comparator uses multiple logic statements comparing four input bits to four output bits, while using three LEDs to indicate greater than, less than, or equal to.

The NAND operation compares each bit in a four bit input with its respective bit in the other four bit input, and returns the result of the two bits ANDed together and then complemented.

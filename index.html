Projects:

Project 1

Motion Activated Gate

Designed and built a motion activated gate using an ESP32 microcontroller, ultrasonic sensor, and servo motor.
Project 2

Pulse Width Modulation

Worked in a group of 3 at RPI to demonstrate PWM control of an LED using an Arduino.
Project 3

Sensor and Alarm System

Worked in a group of 3 at RPI to design and build a circuit that senses change in light (through the use of a photoresistor in a Wheatstone bridge), which is amplified and used to trigger an LED alarm.
Project 4

Analog Synthesizer - VCO

Worked in a group of 3 at RPI to design and build an analog voltage-controlled oscillator (VCO) that is able to produce both a square wave and triangle wave and change frequency.
Project 5

LED Memory Game

Created a call and response game that generates a random sequence of LED flashes that the user has to remember. Four different game modes with LCD screen display. Circuit powered by ESP32 Microcontroller.
Project 6

BASYS Board Calculator

Created a multi-mode switch controlled calculator capable of four bit adding, subtracting (with overflow indicator LED), comparator, and NAND operations. Programmed using VHDL in Vivado on the BASYS 3 board.
Project 7

Logic Circuit / Hazard Corrections

Worked in a group of 3 at RPI to identify and correct hazards in a combinational logic circuit using Karnaugh maps and Boolean algebra.
Project 8

PID Control

Worked in a group of 2 at RPI to program a microcontroller powered car and make corrections using PID control.
